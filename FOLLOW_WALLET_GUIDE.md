# 跟单指定钱包指南

## 🎯 跟单指定钱包流程

### 步骤 1: 分析钱包概况

首先全面了解这个钱包的情况：

```
get_trader_profile(address) → 获取交易者综合分析
```

**示例问题**：
- "分析这个钱包 0x... 的交易表现"
- "这个钱包的盈利情况如何？"

### 步骤 2: 查看当前持仓

了解钱包当前持有哪些市场：

```
get_trader_positions(address) → 获取当前持仓和 PnL
```

**示例问题**：
- "这个钱包当前有哪些持仓？"
- "这个钱包的持仓盈亏情况如何？"

### 步骤 3: 监控交易历史

查看钱包最近的交易行为：

```
get_trader_trades(address, limit: 10) → 获取最近交易
get_trader_activity(address, limit: 20) → 获取完整活动历史
```

**示例问题**：
- "这个钱包最近买了什么？"
- "这个钱包最近 24 小时的交易记录"
- "这个钱包的交易频率如何？"

### 步骤 4: 分析交易策略

深入了解钱包的交易模式：

```
get_trader_profile(address) → 分析策略类型
```

**策略类型识别**：
- 对冲型：同时持有 YES 和 NO
- 方向型：只买 YES 或只买 NO
- 套利型：频繁 split/merge
- 做市型：提供流动性

### 步骤 5: 评估跟单机会

对于钱包买入的市场，评估是否值得跟单：

```
get_market(identifier) → 获取市场信息
get_orderbook(conditionId) → 查看盘口深度
estimate_execution(conditionId, outcome, side, amount) → 预估成交
```

**示例问题**：
- "这个钱包买入的市场现在价格是多少？"
- "如果我现在跟单，预估成交价是多少？"
- "这个市场的流动性如何？"

### 步骤 6: 执行跟单

确认后执行跟单交易：

```
place_limit_order(conditionId, outcome, side, price, size) → 下限价单
place_market_order(conditionId, outcome, side, amount) → 下市价单
```

## 📋 完整跟单流程示例

### 示例：跟单钱包 0x1234...

```
1. 分析钱包概况
   → get_trader_profile(address: "0x1234...")
   结果：盈利 $50,000，胜率 65%，主要策略是方向型

2. 查看当前持仓
   → get_trader_positions(address: "0x1234...")
   结果：持有 5 个市场，总未实现盈利 $5,000

3. 查看最近交易
   → get_trader_trades(address: "0x1234...", limit: 5)
   结果：最近买入市场 A、B、C

4. 分析市场 A
   → get_market(identifier: "market-a-slug")
   结果：市场 A 当前 Yes 价格 0.55

5. 查看盘口
   → get_orderbook(conditionId: "0x...", outcome: "Yes")
   结果：流动性良好，买卖价差 0.01

6. 预估成交
   → estimate_execution(
        conditionId: "0x...",
        outcome: "Yes",
        side: "BUY",
        amount: 100
      )
   结果：预估成交价 0.55，滑点 0.5%

7. 执行跟单
   → place_limit_order(
        conditionId: "0x...",
        outcome: "Yes",
        side: "BUY",
        price: 0.55,
        size: 100
      )
   结果：订单已提交
```

## 💡 跟单策略

### 1. 选择跟单目标

**优秀钱包特征**：
- ✅ 长期盈利稳定
- ✅ 胜率 > 60%
- ✅ 交易风格清晰
- ✅ 风险控制良好

**避免跟单**：
- ❌ 波动性极大
- ❌ 频繁亏损
- ❌ 交易风格不明确
- ❌ 持仓过于集中

### 2. 跟单时机

**最佳时机**：
- 钱包刚买入新市场
- 市场流动性良好
- 价格变化不大
- 市场还有上涨空间

**避免时机**：
- 市场已经大幅波动
- 流动性不足
- 价格已经很高
- 市场即将结束

### 3. 风险管理

- **仓位控制**：不要全仓跟单
- **分散跟单**：跟单多个钱包
- **止损设置**：设置合理的止损点
- **定期评估**：定期检查跟单效果

## ⚠️ 注意事项

1. **价格差异**：钱包买入时的价格可能与你跟单时的价格不同
2. **时机差异**：你看到交易时，市场可能已经变化
3. **策略变化**：钱包的交易策略可能会改变
4. **市场风险**：跟单不能保证盈利，需要自己判断
