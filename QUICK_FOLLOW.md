# 快速跟单指定钱包

## 🚀 一键跟单流程

### 在 Claude Desktop 中直接使用

#### 1. 分析钱包（第一步）

```
"分析钱包 0x你的钱包地址，查看他的交易表现、持仓和最近交易"
```

#### 2. 评估跟单机会（第二步）

```
"这个钱包最近买入的市场，现在价格是多少？我应该跟单吗？"
```

#### 3. 执行跟单（第三步，需要配置钱包）

```
"帮我在这个市场买入 100 USDC 的 Yes，价格 0.55"
```

## 📋 完整示例

### 示例：跟单钱包 0xABC123...

**第一步：全面分析**
```
"帮我分析钱包 0xABC123...：
- 查看他的交易表现
- 分析他的交易策略类型
- 查看他的当前持仓
- 查看他最近 10 笔交易"
```

**第二步：评估跟单**
```
"这个钱包最近买入的市场，帮我：
- 查看市场当前价格
- 分析盘口流动性
- 预估如果跟单 100 USDC 的成交价
- 给出跟单建议"
```

**第三步：执行跟单**
```
"根据分析，帮我在市场 [conditionId] 买入 100 USDC 的 Yes，价格 [价格]"
```

## 🎯 常用命令模板

### 模板 1: 快速分析
```
"分析钱包 [地址] 并推荐跟单机会"
```

### 模板 2: 监控新交易
```
"监控钱包 [地址] 的新交易，发现买入立即告诉我"
```

### 模板 3: 对比分析
```
"对比钱包 [地址1] 和 [地址2]，推荐跟单哪个"
```

### 模板 4: 持仓分析
```
"钱包 [地址] 的持仓中，哪些值得跟单？"
```

## ⚡ 快速开始

直接在 Claude Desktop 中输入：

```
"我想跟单钱包 0x你的钱包地址，帮我分析并推荐跟单机会"
```

Claude 会自动：
1. 分析钱包表现
2. 查看持仓和交易
3. 评估跟单机会
4. 给出建议

---

**提示**：将 `0x你的钱包地址` 替换为实际要跟单的钱包地址
