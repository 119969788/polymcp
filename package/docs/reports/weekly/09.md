## Phase 6: 边缘来源分析

### 核心边缘 (Edge)

**1. Underdog Value Identification (劣势方价值识别)**
```
核心逻辑: 市场倾向于高估热门队伍，低估劣势方
入场条件: 劣势方定价 0.20 - 0.35
预期收益: 胜率 ~35%，但赔率 3:1，期望值为正
```

**2. Spread Mispricing (点差错误定价)**
```
核心逻辑: 大点差 (>10分) 往往过度反映实力差距
入场条件: 点差 > 10 分，买入被让分方
预期收益: 覆盖率高于市场定价
```

**3. 专业知识**
- NBA 专业知识深厚
- 对球队近况、伤病、背靠背等因素有独特理解
- 可能使用量化模型辅助决策

### 边缘弱点

1. **足球市场表现较差**: Man City UCL 大额亏损说明足球不是专长
2. **偏离策略时亏损**: Mavericks vs Kings 高价买入热门导致亏损
3. **无法预测黑天鹅**: 体育比赛有不可预测性

## Phase 7: 可复制策略提取

### 策略 1: NBA Underdog Value Strategy

**核心规则**:
```
市场: NBA 常规赛
方向: 买入劣势方 (Away Underdog)
入场: 定价 0.20 - 0.35 ($20 - $35 每 $100)
头寸: 总资金的 5-10%
退出: 持有到结算
```

**筛选条件**:
- 客场作战的劣势方
- 近期战绩不佳但实力尚可的队伍
- 热门队伍有背靠背或关键球员伤病

**预期表现**:
- 胜率: 30-40%
- 平均赔率: 2.5-4x
- 期望值: +5% to +15%

**核心规则**:
```
市场: NFL 点差盘
```

**筛选条件**:
