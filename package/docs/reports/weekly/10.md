## Phase 7: 可复制的策略

### 策略 1: 冷门狙击

```
条件:
- 目标球队定价 < $0.35 (市场认为胜率 < 35%)
- 你有独立判断认为他们被低估

执行:
- 赛前 1-2 小时建仓
- 初始仓位 = 愿意亏损的金额
- 如果赛中形势有利，可加仓

风险:
- 大部分时候会亏损 (冷门毕竟是冷门)
- 需要大量研究支撑判断
```

### 策略 2: 赛中加仓

```
条件:
- 已有赛前持仓
- 比赛进行中形势向有利方向发展
- 但价格还没有完全反映

执行:
- 使用机器人拆单执行
- 快速扫货，不给市场反应时间

风险:
- 比赛随时可能反转
- 需要实时观看比赛
```

### 策略 3: 不对称对冲

```
条件:
- 对比赛结果有倾向但不确定
- 两边价格都在 $0.45-0.55 附近

执行:
- 在倾向方向押更多 (如 $100K)
- 在反方向押较少 (如 $80K)

结果:
- 倾向方向赢: 大赚
- 反方向赢: 小亏或打平
```
