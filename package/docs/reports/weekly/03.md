## Phase 7: 可复制策略提取

### 策略 1: Underdog Value Betting

**核心逻辑:**
- 在赔率 $0.20-0.45 区间寻找被低估的选项
- 寻找市场共识与实际胜率存在偏差的机会

**执行步骤:**
1. 筛选即将开始的体育赛事（24小时内）
2. 找到赔率低于 $0.45 的一方
3. 评估市场是否过度偏向热门方
4. 如果有信息/分析支持冷门方，大额建仓

**风险提示:**
- 需要体育专业知识
- 高风险，输了就是100%亏损
- 需要较大资金池承受连续失败

---

### 策略 2: Event-Based Momentum

**核心逻辑:**
- 在比赛开始前1-2小时观察市场动向
- 如果有大额资金涌入冷门方，跟随入场

**执行步骤:**
1. 监控即将开始的比赛市场
2. 观察价格变化和交易量
3. 如果冷门方价格开始上涨，跟随买入
4. 使用限价单控制入场价格

**适用场景:**
- NFL/NBA/UFC 等高关注度赛事
- 有实时数据可观察的市场

---

### 策略 3: Multi-Outcome Hedge

**核心逻辑:**
- 同一赛事押注多个相关结果
- 例如: Eagles 赢 + Chiefs 不赢

**执行步骤:**
1. 找到主市场和相关市场
2. 评估联合概率
3. 在两个市场同时建仓
4. 确保任一结果都有正回报
