## Phase 7: 可复制策略提取

### 策略 1: 让分盘价值投资

**原理**: 让分盘价格经常偏离真实概率

**执行方法**:
```
1. 筛选让分盘价格在 0.45-0.55 区间的市场
2. 分析让分点数是否合理
3. 如果认为让分被高估，买入受让方
4. 单笔投入账户 10-20%
```

**关键指标**:
- 让分点数 vs. 历史比分差
- 球队近期进攻/防守效率
- 主客场优势

### 策略 2: 胜负盘+让分盘对冲

**原理**: 利用两个市场之间的价格不一致

**执行方法**:
```
1. 选定一场比赛
2. 分析胜负盘隐含概率
3. 分析让分盘隐含概率
4. 如果存在逻辑矛盾，建立对冲仓位
```

**示例** (Rams vs. Falcons):
- 胜负盘: Rams 75% 胜率
- 让分盘: Rams -7.5，只有 47% 赢盘
- **矛盾**: 如果 Rams 75% 赢，让分盘应该更高
- **操作**: 买 Rams 胜负 + 买 Falcons 让分

### 策略 3: 冷门重仓

**原理**: 被低估的冷门提供正期望值

**执行方法**:
```
1. 筛选价格 < 0.45 的选项
2. 深度分析是否被低估
3. 如果确认，重仓买入
4. 单笔可投入账户 20-30%
```

**关键**: 需要真正的信息优势，不是随便买冷门
