## Phase 7: 可复制的交易策略

### 策略 1: 赛前最后一刻入场

```
时机: 比赛开始前 5-30 分钟
原理:
- 此时伤病/首发信息已公布
- 市场可能还未完全消化信息
- 流动性仍然足够

执行:
1. 监控比赛前 1 小时的新闻
2. 关注关键球员的出场状态
3. 在信息公布后快速入场
```

### 策略 2: Spread 多档位阶梯

```
方法: 同时买入多个 Spread 档位
示例:
- 买 Team -2.5 (保守)
- 买 Team -3.5 (激进)

仓位分配 (改进版):
- 保守档位: 70%
- 激进档位: 30%
(C.SIN 的错误是反过来了)

优势:
- 如果大胜，两个都赢
- 如果小胜，至少保守档位赢
```

### 策略 3: NHL 冰球市场

```
理由:
- NHL 市场效率可能低于 NBA
- 关注度较低，定价错误更多
- C.SIN 在 NHL 表现出色

执行:
1. 专注于 NHL 常规赛
2. 关注东部强队 vs 弱队的对决
3. Lightning、Hurricanes、Panthers 等队值得关注
```

### 策略 4: 客场劣势方机会

```
观察:
- C.SIN 多次押注客场劣势方 (赔率 < 0.50)
- 例如 Heat 客场 vs Nuggets

逻辑:
- 市场可能过度看好主场优势
- 某些客场强队被低估

执行:
1. 识别被低估的客场球队
2. 在赔率 0.45-0.48 区间入场
3. 设置止损（如赔率跌破 0.40）
```

## Phase 8: 风险分析

### 主要风险

| 风险类型 | 描述 | C.SIN 案例 |
|---------|------|-----------|
| **集中度风险** | 单笔仓位过大 | Trail Blazers -3.5 单笔 $282K 亏损 |
| **档位选择风险** | 激进档位占比过高 | -3.5 投入是 -2.5 的 2.8 倍 |
| **信息风险** | 信息优势可能不总是正确 | 即使知道 Trail Blazers 会赢，也无法确定赢多少 |
| **流动性风险** | 大额订单推高价格 | 多笔交易中可见价格逐步上升 |

### 潜在问题

1. **100% 胜率不可持续**: 50 连胜是统计异常，未来必然会有亏损
2. **单笔亏损可吞噬多笔盈利**: Trail Blazers 单笔 -$282K 相当于 2 笔平均盈利
3. **策略容量有限**: 大额交易会推高价格，影响入场成本
